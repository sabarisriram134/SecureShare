# SecureShare – Decentralized Encrypted File Sharing (AES + IPFS + Ethereum)

A complete end-to-end secure file storage & sharing system using:

- Node.js backend  
- AES-GCM encryption  
- IPFS (Pinata/Web3.storage)  
- Solidity smart contract  
- React + Vite  
- MetaMask wallet  
- Access control + audit logs  

This project is fully ready for college submission.
